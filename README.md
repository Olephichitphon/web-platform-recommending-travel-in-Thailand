## 🇹🇭 ภาษาไทย

แพลตฟอร์มเว็บไซต์สำหรับแนะนำการท่องเที่ยวในประเทศไทย ที่นำเสนอประสบการณ์ที่น่าจดจำผ่านสถานที่ท่องเที่ยวที่น่าสนใจ แพ็กเกจทัวร์สุดคุ้ม และบล็อกเรื่องราวต่างๆ พร้อมรองรับหลายภาษา

### 🌟 จุดเด่นของโปรเจกต์ (Features)
- **🌍 รองรับหลายภาษา (Multilingual):** รองรับภาษาไทย (TH), ภาษาอังกฤษ (EN), และภาษาจีน (CN)
- **📱 Responsive Design:** แสดงผลได้อย่างสวยงามทุกหน้าจอ ไม่ว่าจะเป็นคอมพิวเตอร์ แท็บเล็ต หรือมือถือ
- **✨ Smooth Animations:** มีการใช้ Framer Motion เพิ่มแอนิเมชันให้กับการเลื่อนหน้าจอและการแสดงผลส่วนต่างๆ
- **🗺️ เนื้อหาครบถ้วน:** ประกอบด้วยส่วนค้นหาทริป, จุดหมายยอดนิยม, ข้อเสนอพิเศษ, บล็อกท่องเที่ยว, รีวิวจากลูกค้า, และส่วนติดต่อเรา
- **🖼️ Interactive Modals:** มี Popup รายละเอียดทัวร์และบล็อกให้อ่านเพิ่มเติมได้ทันทีโดยไม่ต้องเปลี่ยนหน้า

### 💻 เทคโนโลยีที่ใช้ (Tech Stack)
- **Frontend Framework:** React 19 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

### 🚀 วิธีการติดตั้งและรันโปรเจกต์

**สิ่งที่ต้องมี:** Node.js 

1. โคลนโปรเจกต์และเข้าไปที่โฟลเดอร์:
   ```bash
   git clone <your-repo-url>
   cd thaitravel---travel-in-thailand
   ```

2. ติดตั้ง Dependencies:
   ```bash
   npm install
   ```

3. รันโปรเจกต์ในโหมด Development:
   ```bash
   npm run dev
   ```

4. เปิดเบราว์เซอร์ไปที่ลิงก์ที่แสดงใน Terminal (ปกติจะเป็น `http://localhost:3000`)

## 🇬🇧 English

A web platform recommending travel in Thailand, presenting memorable experiences through interesting tourist attractions, value tour packages, and travel blogs, with full multilingual support.

### 🌟 Features
- **🌍 Multilingual Support:** Supports Thai (TH), English (EN), and Chinese (CN).
- **📱 Responsive Design:** Beautifully displayed on all screens, including desktops, tablets, and mobile devices.
- **✨ Smooth Animations:** Utilizes Framer Motion for smooth scrolling and section reveal animations.
- **🗺️ Comprehensive Content:** Includes Trip Search, Popular Destinations, Special Offers, Travel Blogs, Customer Testimonials, and Contact sections.
- **🖼️ Interactive Modals:** Features popups for tour details and blogs for quick reading without page reloads.

### 💻 Tech Stack
- **Frontend Framework:** React 19 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

### 🚀 Run Locally

**Prerequisites:** Node.js 

1. Clone the project and navigate into the directory:
   ```bash
   git clone <your-repo-url>
   cd thaitravel---travel-in-thailand
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the project in development mode:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the link shown in the terminal (usually `http://localhost:3000`).
